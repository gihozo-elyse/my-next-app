##  Getting Started

### Prerequisites
- Node.js 14.6.0 or later
- npm or yarn

### Installation
```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
npm start
```

##  Features
-  Dark/Light mode
-  Responsive design
-  Fast page loads with different rendering techniques
-  Blog with categories
-  Real-time updates
